School Management System

Project Overview

The School Management System is a web-based application designed to manage student attendance, exam results, school activities, communication, and other academic operations efficiently. The system provides separate dashboards for Admin, Teachers, Students, and Parents.

Admin Panel Features

  • Managing user accounts (Teacher, Student, Parent)
  • Managing classes and subjects
  • Managing class routine
  • Managing exam marks
  • Managing student attendance
  • Managing school events and activities
  • Managing library (Books, Borrowing Records)
  • Publishing notices

Teacher Panel Features

  • Managing students
  • Managing exam marks
  • Managing attendance

Student Panel Features

  • View class routine
  • View exam marks
  • View attendance

Parent Panel Features

  • View children's marks
  • View children's payment invoices
  • View children's class routine
  • Messaging with teachers
